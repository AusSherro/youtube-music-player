Add a variable to track current duration (needed for seek calculation):

In the existing `onMetadataUpdate` callback, after updating play/pause icon but still inside the `if (metadata)` block, add progress bar update:

```typescript
// Update progress bar (D-04, D-06)
currentDuration = metadata.duration
if (metadata.duration > 0) {
  const pct = Math.min((metadata.progress / metadata.duration) * 100, 100)
  progressFill.style.width = `${pct}%`
} else {
  progressFill.style.width = '0%'
}
```

In the `else` block (no metadata), reset progress:
```typescript
progressFill.style.width = '0%'
currentDuration = 0
```

Add click-to-seek handler after the playback control listeners:

```typescript
// Progress bar click-to-seek (D-05)
progressContainer.addEventListener('click', (e) => {
  if (currentDuration <= 0) return
  const rect = progressContainer.getBoundingClientRect()
  const ratio = Math.max(0, Math.min(1, (e.clientX - rect.left) / rect.width))
  const seekSeconds = ratio * currentDuration
  window.electronAPI.seek(seekSeconds)
})
```

Add navigation-away handler at the end of the file (before `export {}`):

```typescript
// Navigation-away detection (D-10)
window.electronAPI.onNavigationState((onYTM) => {
  if (onYTM) {
    navAway.classList.add('hidden')
  } else {
    navAway.classList.remove('hidden')
  }
})
```
